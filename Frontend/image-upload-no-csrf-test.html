Django Configuration Check

If image upload works without CSRF but fails with CSRF, the issue is Django CSRF configuration.

Add this to your Django settings.py:

CSRF_TRUSTED_ORIGINS = [
    'http://127.0.0.1:5500',
    'http://localhost:5500',
]

# For testing only, you can temporarily disable CSRF:
# CSRF_COOKIE_SECURE = False
# CSRF_COOKIE_HTTPONLY = False
